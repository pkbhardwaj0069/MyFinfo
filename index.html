Google Web Client ID Guide

  1. Go to Google Cloud Console.
  2. Create a project (e.g. My Finfo).
  3. Go to APIs & Services > Library, search for Gmail API, click Enable.
  4. Go to OAuth consent screen, select External, and fill in mandatory fields. Under Scopes, add .../auth/gmail.readonly. Add your email as a Test User.
  5. Go to Credentials > Create Credentials > OAuth client ID.
  6. Select Web application as the application type.
  7. Under Authorized JavaScript origins, add your local app URI: http://localhost:5500 (or your current web server address).
  8. Click Create, copy the Client ID, paste it above, and click Save.
